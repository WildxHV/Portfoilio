Blogza

This page contains the case study of Blogza: A Blogging Website, which includes the Project Overview, Tools Used and Live Links to the official product.

Project Overview

Developed a blog website using Django-Web framework, HTML, CSS, and JavaScript. The website allows users to create, edit, and delete posts. The homepage lists all the categories and blog posts, and there is a dedicated detail page for each individual post. TinyMCE was also incorporated into the website as a text rich editor for adding new blog posts to the site. The website was deployed on Heroku.

A captivating blog website, where one can explore a diverse range of articles on various topics. The platform offers an engaging community for viewers to enjoy insightful discoveries, as our team of owners curates and shares valuable content with you.
